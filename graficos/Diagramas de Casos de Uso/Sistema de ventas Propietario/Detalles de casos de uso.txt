Detalle de caso de uso - Administrar productos:

Actores involucrados:
- Propietario

Descripción:
El propietario gestiona los productos disponibles en el sistema.

Flujo principal de eventos:
1. El propietario selecciona la opción "Administrar productos" en el sistema.
2. El sistema muestra el listado de productos y las opciones de gestión disponibles.
3. El propietario puede agregar un nuevo producto, modificar un producto existente o eliminar un producto.
4. El sistema actualiza la información de los productos según las acciones realizadas
